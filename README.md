# Hosted URL - https://sandeep64s.github.io/HabitTracker/

This is a Habit Tracker  Project which is built using Nodejs, Ejs , CSS, Javascript , Bootstrap


## Features
- Add Habit <br/>
- Each Habit Track Weekly<br/>
  - Mark Habit as Done
  - Mark as Undone
  - Toggle to Default - No action setting
  - Delete a habit
